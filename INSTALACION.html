🚀 PATERSAY Website - Guía Completa de Instalación

✅ Felicidades! Has recibido la estructura completa del sitio web PATERSAY basado en el diseño Minimal Executive 4.1

📦 Contenido Descargado

La carpeta patersay-website/ contiene:

  • index.html - Página principal con todas las secciones
  • css/styles.css - Estilos responsivos (desktop, tablet, mobile)
  • js/main.js - JavaScript funcional (búsqueda, modales, WhatsApp)
  • README.md - Documentación completa
  • ESTRUCTURA.md - Guía de carpetas y próximos pasos
  • package.json - Configuración npm
  • .env.example - Variables de entorno
  • .gitignore - Configuración Git

🎯 Paso 1: Descargar la Carpeta

Si aún no lo has hecho, descarga la carpeta patersay-website completa en tu computadora.

🖥️ Paso 2: Abrir en VS Code

Opción A: Desde Terminal

cd ruta/a/patersay-website
code .

Opción B: Desde VS Code

  1. Abre VS Code
  2. File → Open Folder
  3. Selecciona la carpeta patersay-website
  4. Click en "Select Folder"

⚙️ Paso 3: Instalar Extensiones (Recomendado)

En VS Code, instala estas extensiones para mejor desarrollo:

  • Live Server - Para ver cambios en tiempo real
  • Prettier - Formateador de código
  • HTML to CSS autocompletion - Autocompletado CSS
  • Thunder Client o Postman - Para testing de APIs (futuro)
💡 Tip: En VS Code, presiona Ctrl+Shift+X (Windows/Linux) o Cmd+Shift+X (Mac) para abrir la tienda de extensiones.

🌐 Paso 4: Ver la Página en el Navegador

Con Live Server (Recomendado)

  1. Click derecho en index.html
  2. Selecciona "Open with Live Server"
  3. Se abrirá automáticamente en http://localhost:5500
  4. Los cambios se reflejan en tiempo real

Sin Live Server (Manual)

  1. Haz doble clic en index.html
  2. Se abrirá en tu navegador predeterminado
  3. Para ver cambios, recarga la página (F5)
⚠️ Nota: Algunos navegadores no permiten características de JavaScript con file:// local. Usa Live Server para evitar problemas.

🔧 Editar Contenido

Cambiar Texto Principal

Abre index.html y busca los textos. Por ejemplo:

<h1>Soluciones tecnológicas de impacto</h1>

Cámbialo por tu texto y guarda (Ctrl+S o Cmd+S).

Cambiar Colores

Abre css/styles.css y busca #0066cc (azul corporativo):

.btn-primary {
    background: #0066cc;  /* Cambia aquí */
}

Agregar Imágenes

  1. Crea carpeta patersay-website/assets/images/
  2. Coloca tus imágenes (hero.jpg, etc.)
  3. En HTML, actualiza la ruta: <img src="assets/images/hero.jpg" alt="Description">

  • Dispositivos específicos: iPhone 12, iPad, Android

🐛 Troubleshooting

❌ "Live Server no funciona"

Soluciones:

  • Asegúrate de tener Live Server instalado
  • Reinicia VS Code
  • Verifica puerto 5500 no esté ocupado
  • Abre manualmente en navegador

❌ "Los estilos no se ven"

Soluciones:

  • Verifica que css/styles.css exista
  • Recarga la página (Ctrl+Shift+R)
  • Limpia caché del navegador
  • Abre DevTools (F12) → Pestaña Network para verificar recursos

❌ "JavaScript no funciona"

Soluciones:

  • Verifica que js/main.js exista
  • Abre DevTools (F12) → Pestaña Console para ver errores
  • Revisa que no haya errores de sintaxis
  • Recarga la página
